# Installation Guide

## Prerequisites

- ServiceNow Personal Developer Instance (PDI)
- System Administrator role
- Git
- GitHub

## Setup Steps

1. Clone the repository.
2. Import the application/update set.
3. Activate required plugins:
   - Flow Designer
   - Virtual Agent
4. Configure the SLA.
5. Import Virtual Agent topics.
6. Configure notifications.
7. Verify dashboard and reports.

## Verification

- Create a test incident.
- Confirm SLA starts.
- Verify warning notifications.
